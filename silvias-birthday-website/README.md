# Silvia's Birthday Website

This project is a beautiful and romantic birthday website dedicated to my girlfriend, Silvia, who is turning 17. The website features a lovely design, heartfelt messages, and interactive elements to celebrate her special day.

## Project Structure

```
silvias-birthday-website
├── index.html        # Main HTML document for the website
├── css
│   └── style.css     # Styles for the website
├── js
│   └── main.js       # JavaScript for interactivity
├── assets
│   ├── images        # Images used on the website
│   └── fonts         # Custom fonts for typography
├── .gitignore        # Files and directories to ignore in version control
└── README.md         # Documentation for the project
```

## Features

- **Romantic Design**: The website is designed with a romantic aesthetic, using soft colors and elegant fonts.
- **Interactive Elements**: JavaScript is used to add interactivity, such as animations and dynamic content updates.
- **Personal Touch**: The website includes personal messages and images to make Silvia feel special on her birthday.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. Customize the content in `index.html`, `css/style.css`, and `js/main.js` as needed.
